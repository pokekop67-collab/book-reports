🔑 獨家發現：作者的建書心智模型 🎤 現場

簽書會中，小竺首次公開他「從一個念頭到一本書」的完整思考過程。這是書本身不會告訴你的 meta-level 洞見。

核心架構：點 → 線 → 面 → 體

整本書的建構邏輯不是「章節列表」，而是一個從原點向外擴展的立體結構。

⚫ 原點（點）：KISS 高價值表達模型

一切從這個信仰開始 — Key Message + Information + Structure + Skill。這不只是書的框架，更是作者看待所有表達行為的核心信念。

📏 連點成線：消費決策旅程 5 階段

從 KISS 延伸出路徑思維 — 客戶的決策不是一瞬間的事，而是：產生需求 → 確認類別 → 選擇品牌 → 選擇商品 → 締結成交。每個階段的溝通重點完全不同。

引發痛點→ 教育市場→ 凸顯優勢→ 降低風險→ 強化信任

🔲 織線成面：精準表達三層框架

面 = 有範圍的思考。精準表達不是「把話說精簡」，而是三層同時到位：

層次核心問題書中位置
Why to say你期待這次對話達成什麼結果？（兼顧對方的期望值）觀念篇 Ch4-5
What to say什麼訊息能戳到痛點或搔到癢處？觀念篇 Ch3, Ch6
How to say用什麼策略和技巧傳遞？實戰篇全部

加上四階段精準表達層次（現場獨家）：目標清晰 → 框架明確 → 邏輯嚴謹 → 內容有重點

📦 構面成體：工具與模組的堆疊

在「面」的基礎上，堆疊具體可用的思考工具和表達模組 — 自我介紹模組、提問模組、異議處理模組、結單模組等。這就是實戰篇的核心價值：即學即用的工具箱。

💎 超級大腦讀書法洞見：

這個「點線面體」不只是書的架構，更是作者建議所有人組織表達內容的方法。先找到核心信念（點），沿著客戶旅程展開路徑（線），建立精準表達的範圍（面），最後用工具和模組讓它立體化（體）。書本身就是方法論的最佳範例。

🗺️ 實戰篇全貌（簽書會獨家揭露）🎤 現場

試閱版只有觀念篇 Ch1-6，但簽書會完整揭露了實戰篇的單元結構。

單元核心任務對應「點線面體」
消費決策旅程切出客戶從無需求到成交的路徑線（路徑）
自我介紹（Ch14）ABCDE 模組：角色定位→獨特能力→目標客群→關鍵問題→商業邀請體（模組）
洞察需求提問工具，挖掘客戶真正的內在需求體（工具）
價值表達如何讓客戶感受到你的獨特價值體（策略）
成交/結單異議處理、結單技巧體（模組）
聆聽的藝術用耳朵、眼睛、身體、聲音、筆記、心、腦聆聽體（心法）
讚美不說話也重要的表達行為體（技巧）

🕸️ 全域交叉比對：隱藏連結網絡 🔗 交叉

1. 書中沒寫但現場親口說的核心信念

信念①「決策是100%感性的」

小竺原話：
思考是理性的，但決策一定是感性的。有很多醫學跟科學證據證明，情緒中樞受損的人，他完全不會影響思考，但會影響決策。

這解釋了為什麼他把 Information 拆成「資訊、觀點、感受」三種成分 — 資訊建立信任，觀點展現獨特性，感受驅動決策。三者缺一不可，但比例要根據對方狀態動態調整。

對 Rex 的啟示：影片提案時，不要只給客戶「規格和價格」（資訊），要加入「為什麼這個方向對他最好」（觀點）和「如果不做這件事，可能會怎樣」（感受）。

信念②「教育市場 ≠ 教訓市場」

現場學員反思：
我以前常常講太艱深的內容，最後讓人家感覺你是不是一直覺得我現在做自己不好。我沒有想要教訓市場的意圖，但最後我這麼做了。

這是書中完全沒有提到的概念，但在現場引發強烈共鳴。教育是「讓對方感覺被賦能」，教訓是「讓對方感覺被否定」。同樣的專業知識，表達方式決定了對方接收到的是教育還是教訓。

信念③「半推半就，弄假成真」— 作者的反目標論

小竺原話：
我不是一個非常目標導向的人。可是我就是有點像半推半就弄假成真 — 機會來了剛好遇到了，問題發生了我得找機會。

這與書中強烈的「以終為始」論述形成微妙張力。書教你「設定清晰目標」，但作者自己的成功路徑卻是「看見訊號 → 回應機會 → 累積成系統」。這揭示了一個隱藏元原則：表達需要目標，但人生可以靈活。

2. 現場教學展示的隱藏方法論

🎯 ABCDE 自我介紹模組（Ch14 精華版）

我是A（角色定位），擅長B（獨特能力），協助C（目標客群）解決D（關鍵問題），需要E（商業邀請）。

作者的示範：「我是高言值表達力教練小竺，也是中翻中的專家。我擅長運用系統化、模組化與深入淺出的教學方式，協助有企圖心的專業工作者，將抽象的思維轉化為有價值的商業語言。」

E 的兩種方向：「我要的」（介紹客戶給我）或「我能給的」（我介紹專業人士給你）— 兩種都是給對方一個「找你的理由」。

🎯 消費決策旅程 × KISS = 不同階段說不同話

階段客戶狀態溝通重點常犯錯誤
產生需求沒意識到問題引發痛點、願景描繪一上來就講產品優勢
確認類別知道要解決，不確定方式教育市場、差異化定位否定對方現有選擇
選擇品牌比較各家凸顯獨特優勢、案例只講價格
選擇商品決定要買降低風險、試用體驗繼續推銷
締結成交準備付款信任強化、轉介機制忘了售後跟進

🎯 四種反應診斷法的現場展演

對方反應問題出在現場補充
😶 沒反應Skill（技巧）太小聲、太平淡、太 boring
😕 蛤？Structure（結構）講到什麼才想什麼，邊講邊想
🤷 所以咧？Information（訊息）沒重點、沒 CTA、跟對方無關
🙅 認同但不行動Key Message沒戳到痛點，價值感不夠強烈

現場金句：「每個潛在客戶心裡都有四個字 —『關我屁事』。」

3. 表面矛盾組（書 vs 現場）

矛盾組：「反對話術」vs「提供大量模組」

書中立場：不相信話術，每次對話無法複製貼上。

現場立場：提供了 ABCDE 自我介紹、提問模組、結單模組等大量「公式」。

深層統一（現場原話解答）：

「用模組的目的是引導你思考，不是取代你的大腦。如果你有更好的想法，你可以不用模組。但如果你現在腦袋一片空白，模組可以幫助大家。」

→ 模組是「訓練輪」，不是「永久義肢」。確保狀況再差都有60分的表現。

矛盾組：「邊想邊講 OK」vs「先想後說」

書中：「脫口而出是本能，謹言慎行是本事」— 暗示要先想好再說。

現場：「邊想邊講本來就是正常的過程，因為大部分對話是 real-time 發生的。」

深層統一：

關鍵不是「想完才說」，而是「想在前面」。「邊想邊講」是OK的（想→講→想→講），但「邊講邊想」是危險的（講→才開始想），因為第一句話會定錨後續所有思緒。

💎 元原則萃取（雙源升級版）

元原則 01「你現在的表達方式，是你過往人生的總和」

這句話在現場被強調了三次以上。它的深層含義是：改變表達不是學一個技巧，而是覺察並調整根深蒂固的習慣。如果不理解這一點，就會覺得「太麻煩了，不好改，這就不是我」。

元原則 02「精準表達不是把話說精簡，是能不能達成我的期待」

話多話少、頁面多少、文字長短都不是重點。重點是有沒有辦法達成目標。這個元原則統一了書中所有的工具和方法 — 一切以目標為導向。

元原則 03「不處理、不改變，會更痛」

這是作者在談消費決策旅程「產生需求」階段時揭露的底層信念。需求不一定是天然存在的，可以被「發掘」甚至「創造」— 讓對方看到不行動的代價。

元原則 04「種子心態：不急著收割，先建立信任」

多次在現場展現：「我先撒下種子，什麼時候發酵我不知道。我很清楚展現我的價值，讓對方理解我可以協助什麼。」這與「以終為始」不矛盾 — 目標是長期信任，不是即時成交。

🔮 Rex 的個人化透鏡（雙源升級版）

轉化 1：你在簽書會的自我介紹 → ABCDE 重構

你的現場版本：
「我是 Rex，摄影師，拍過總統跟跨國企業。影像上有問題都可以找我。」

用 ABCDE 重構：

我是叡思影像的創辦人 Rex（A），擅長用影像策略為企業傳遞品牌核心價值，而不只是拍出漂亮的畫面（B）。我協助需要舉辦高規格活動或製作品牌形象影片的中大型企業（C），解決「影像做了卻無法精準傳達品牌訊息」的問題（D）。如果你身邊有企業正在規劃品牌影片或重要活動記錄，歡迎介紹給我，我可以從影像顧問的角度先幫他們釐清方向（E）。

差異：原版停在「我是什麼」，重構版延伸到「我能為你解決什麼」+ 給對方一個找你的理由。

轉化 2：消費決策旅程 → 叡思影像的客戶路徑設計

階段Rex 的溝通重點具體動作
產生需求讓企業意識到「沒有好影像 = 品牌無法被感受」社群內容：分享「影像做對 vs 做錯」的案例對比
確認類別教育市場：為什麼要找專業團隊而非自己拍顧問式內容：「影片製作前要問自己的3個問題」
選擇品牌凸顯你的獨特優勢（政商經驗、策略思維）案例展示：從策略到成品的完整思考過程
選擇商品降低風險，提供試用免費30分鐘影像顧問諮詢
締結成交信任強化 + 轉介機制完成後提供精華剪輯，方便客戶內部展示成果

轉化 3：「客戶不理你」→ 精準診斷

簽書會中你提到「客戶不太理你」，小竺當場做了一個示範性的重新定義：「不理你」其實是「沒有買單你的建議」— 這兩個的應對方式完全不同。

如果是「聽都不想聽」→ 信任不足，需要先建立關係（決策旅程前端）。
如果是「聽了但不採納」→ Key Message 沒戳到痛點，需要先了解對方在意什麼。

行動建議：下次遇到這種情況，先問自己：「客戶現在在決策旅程的哪個階段？我講的話是這個階段該講的嗎？」

轉化 4：資訊 × 觀點 × 感受 → 保險業溝通升級

小竺在現場強調：「如果你完全沒有資訊，全是觀點和感受，聽起來就是靠北。如果只有資訊沒有觀點，完全沒有獨特性。」

保險場景應用：

❌ 純資訊：「這份保單年繳3萬，保障額度500萬。」

❌ 純觀點：「我覺得你應該要買這份保單。」

✅ 三者兼具：「根據你的家庭狀況，如果主要收入來源中斷，家庭支出至少需要維持3年（資訊）。我認為用每月2500元的成本換取500萬的保障，對你來說是性價比最高的選擇（觀點）。說實話，我自己也是因為成為爸爸之後，才真正理解這份安全感有多重要（感受）。」

🔍 缺席分析（雙源升級版）

缺席 1：非同步表達（文字、社群、影音腳本）

整本書 + 簽書會都聚焦在「即時口語溝通」場景。但一人公司經營者的大量表達是非同步的（社群貼文、Email、提案文件）。作者在現場說「文字也是一種說法，只是以文字方式呈現」，但並未展開方法論。

缺席 2：內向者的表達策略

現場互動以外向式場景為主（上台演講、一對多、活動社交）。對於內向型專業工作者，如何在不強迫自己變外向的前提下，運用 KISS 框架建立影響力？書中未特別討論。

缺席 3：數位時代的「不見面表達」

現場學員提到的 SEO、網站內容等場景，其實就是「不見面的表達」。KISS 框架理論上適用，但實際轉化需要額外思考（例如：線上的 Structure 和線下的 Structure 規則不同）。

🎬 行動轉化（雙源升級版）

📌 本週啟動

  • 用 ABCDE 重寫你的自我介紹，30秒內能講完
  • 在下一次客戶提案前，判斷客戶在「決策旅程5階段」的哪個位置
  • 檢視你最近的社群貼文，有沒有同時包含「資訊＋觀點＋感受」三種成分

📌 本月實驗

  • 設計叡思影像的消費決策旅程，每個階段的溝通內容都不同
  • 每次重要對話後用「四種反應診斷法」自我檢測：對方是沒反應？蛤？所以咧？還是認同但不行動？

📌 長期追蹤

  • 「種子轉化率」— 你撒下的種子（自我介紹、社群內容、諮詢服務），多少最終發芽成客戶？追蹤3-6個月的回流數據
